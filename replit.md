# Basket Tournament Scheduler

## Overview
A simple static web application for scheduling basketball tournament matches using a Greedy Fair-Rest Algorithm. The app is written in Indonesian.

## Project Structure
- `index.html` - Single-page application with embedded CSS and JavaScript

## Features
- Configure tournament settings (number of groups, teams per group, start/end times, match duration)
- Generates tournament schedule automatically
- Shows tournament statistics (total matches, matches per day, etc.)

## Technology
- Pure HTML/CSS/JavaScript (no build system required)
- Static file serving with `npx serve`

## Running Locally
The project runs on port 5000 using `npx serve -l 5000 .`

## Deployment
Configured for static deployment serving the root directory.
